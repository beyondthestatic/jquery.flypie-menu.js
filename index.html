It's Easy and Smart

FlyPie-Menu.js is easy to setup and configure. Try it in your project.

Browser Support

This plugin was was tested and works on Chrome, Safari and Firefox.

Foolproof Implementation

$('.flypie').flypiemenu();

Simple to adjust

$('.flypie').flypiemenu({
	'degreeStart': 192, 
	'maxItems': 8, 
	'itemRad': 35, 
	'orbitRad': 89, 
	'buttonRad': 22,
});

Language Selection

Live Demo

Click on the big flag to open and close the FlyPie-Menu. Click on an Item to change the Language/Flag.

JavaScript

$('#toggleLanguageMenu').flypiemenu({
  'degreeStart':  0,
  'maxItems':     8,
  'itemRad':      25,
  'orbitRad':     80,
  'buttonRad':    35,
  'orientation':  'top-right',
  'popup':        'false',
  'mode':         'click',
  'direction':    'counterclockwise'
});

$('.flag').click(function(e) {
  var activeImg = $(this).find('img');
  $('#toggleLanguageMenu img').attr({
    'src':     activeImg.attr('src'),
    'title':   activeImg.attr('title'),
    'alt':     activeImg.attr('alt'),
  });
});

        <img src="img/flags/de.png"
             title="German"
